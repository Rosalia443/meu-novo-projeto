# Gerador de CV

## Estrutura de pastas

- **css:** Arquivos CSS (folhas de estilo que formatam cada um dos modelos de currículo);
- **img:** Arquivos de imagem (ícones que aparecem nos currículos);
- **js:** Contém o arquivo JavaScript:
    - **index.js:** Arquivo JavaScript responsável por adicionar e remover formações e experiências dinamicamente (código comentado);
- **modelos:** Arquivos PHP que representam o modelo disponivel;
    - **modelo.php:** Modelo moderno preto;
    **-layout:**Arquivos PHP que armazenam cabeçalho e rodapé ;
- **gerarCV:** Arquivo PHP que recebe os dados do formulário e imprime no modelo de currículo selecionado (código comentado);
- **index.php:** Arquivo inicial da aplicação, que contém uma tela de apresentação.
-**contato.php:**Arquivo de formulario 
## Tecnologias utilizadas

- HTML;
- CSS;
- JavaScript;
- jQuery;
- Bootstrap;
- PHP.
